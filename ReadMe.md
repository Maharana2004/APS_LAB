# APS Lab

## Overview
This repository contains experiments and simulations developed for the APS Lab course.  
It provides hands-on exercises to understand applied physics concepts through computational models and practical implementations.

## Features
- Simulation scripts for mechanics, electromagnetism, and thermodynamics
- Data visualization tools for lab results
- Example datasets for practice
- Modular code structure for easy extension
